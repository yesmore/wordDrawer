### 应用概述

---

**wordDrawer：一款轻量级桌面词典软件，使用Electron-vue、开放Api开发，零服务端，开箱即用。**

wordDrawer支持以下环境：

- Windows

  

### 特色功能

---

wordDrawer常用功能：

- 翻译

- 缩写查询
- 新华字典
- 成语查询
- 文章关键词提取
- 文章摘要提取
- 网络热词

### 开发说明

---

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


# lint all JS/Vue component files in `src/`
npm run lint

```
