## 应用概述

**wordDrawer：一款轻量级桌面词典软件，使用Electron-vue、开放Api开发，零服务端，开箱即用。**

wordDrawer支持以下环境：

- Windows


## 特色功能

wordDrawer常用功能：

- 翻译
- 缩写查询
- 新华字典
- 成语查询
- 文章关键词提取
- 文章摘要提取
- 网络热词

## 开发说明

如果你想要学习、开发、修改或自行构建 wordDrawer，可以依照下面的指示：

1. 你需要有Node、Git、Vue-Cli环境，了解 npm 相关知识。

2. `git clone https://github.com/yesmore/wordDrawer.git` 并进入项目。

3. `npm install` 下载依赖。
